# raspberry-showip
Lxpanel plugin to show eth0 IP address on top bar

# Installation
Compile source code (directly on Raspberry Pi) => showip.so is produced
Paste it in /usr/lib/arm-linux-gnueabihf/lxpanel/plugins/showip.so
